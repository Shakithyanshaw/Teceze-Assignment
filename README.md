📌 Employee Management System (MERN Stack)

A full-stack Employee Management System built using the MERN stack (MongoDB, Express, React, Node.js).
This project allows user authentication and employee CRUD operations with a clean UI and REST API backend.

🚀 Features

👨‍💼 Add new employees
✏️ Edit employee details
🗑️ Delete employees
📋 View all employees in table format
⚡ REST API integration with backend
🎯 Responsive UI

🛠️ Tech Stack

Frontend:

React.js
React Router
Axios
Tailwind CSS

Backend:

Node.js
Express.js
MongoDB
Mongoose
